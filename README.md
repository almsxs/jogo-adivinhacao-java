# 🎮 Jogo de Adivinhação em Java

Projeto simples feito em Java com interface gráfica usando Swing.

## 💡 Como funciona
O sistema gera um número aleatório entre 1 e 100, e o usuário precisa adivinhar.

## 🧠 Tecnologias usadas
- Java
- Swing (interface gráfica)

- ## ▶️ Como executar

1. Abra o projeto em uma IDE (VSCode ou IntelliJ)
2. Execute a classe `JogoTop.java`
3. O jogo será iniciado com interface gráfica
